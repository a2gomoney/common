DEXT5 Editor :: KeepTableWidthLikeWord

XML : <keep_table_width_like_word>.....</keep_table_width_like_word>, Config : config.KeepTableWidthLikeWord

Config 설정이 우선 적용됩니다.
테이블의 좌/우 셀 추가를 할 경우 테이블의 width가 Word의 동작과 동일하게 조정되도록 설정합니다.

remarks

기본값은 "0"이고 "1"로 설정 시 테이블의 좌/우 셀 추가를 할 경우
테이블의 width가 Word의 동작과 동일하게 조정됩니다.

sample code

테이블의 좌/우 셀 추가를 할 경우 테이블의 width가 Word의 동작과 동일하게 조정되도록 설정합니다.

XML

.....
<setting>
    .....
    <table_property>
        .....
        <keep_table_width_like_word>1</keep_table_width_like_word>
        .....
    </table_property>
    .....
</setting>
.....
