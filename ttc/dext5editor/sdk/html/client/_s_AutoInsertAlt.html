DEXT5 Editor :: AutoInsertAlt

XML : <auto_insert_alt>.....</auto_insert_alt>, Config : config.AutoInsertAlt

Config 설정이 우선 적용됩니다.
이미지 삽입 시 img 태그에 alt 속성을 넣어 줍니다.

remarks

기본 값은 "1"이며 이미지 삽입 시 img 태그에 alt 속성을 넣어 줌,
"0"으로 설정 시 img 태그에 alt 속성을 넣어 주지 않도록 합니다.

sample code

이미지 삽입 시 img 태그에 alt 속성을 넣어 주도록 합니다.

XML

.....
<setting>
    .....
    <auto_insert_alt>1</auto_insert_alt>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.AutoInsertAlt = "1";

        new Dext5editor("editor1");

    </script>       
</div>
