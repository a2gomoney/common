DEXT5 Editor :: ListSpaceSize

XML : <list_space_size>.....</list_space_size>, Config : config.ListSpaceSize

Config 설정이 우선 적용됩니다.
글머리 번호/기호 생성시 생성 간격을 설정합니다.

remarks

기본값은 "40px" 입니다.
값 + "단위"를 입력하여 설정합니다.
숫자만 입력시 px로 적용됩니다.

sample code

글머리 번호/기호 생성시 생성 간격을 30pt로 설정합니다.

XML

.....
<setting>
    .....
    <list_space_size>30pt</list_space_size>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>
