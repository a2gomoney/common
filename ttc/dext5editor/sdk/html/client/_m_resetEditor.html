DEXT5 Editor :: resetEditor

void resetEditor(resetAll, skipConfirm, editor_id)

에디터 작성내용 및 추가된 css, js, title 등을 초기화 합니다.

return value

없음.

parameters

resetAll  추가된 css url, js url, title 의 초기화 여부 입니다.
skipConfirm  동작 시 작성영역에 내용이 있을 경우 사용자에게 confirm 창을 노출할지 결정하는 옵션입니다.
editor_id  동작될 에디터의 id 입니다.

remarks

없음.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function resetFunction(editorID){
        // 에디터 작성영역 및 title을 초기화 하고 추가된 css, js url을 모두 제거합니다.
        // 작성영역에 내용이 있어도 사용자에게 confirm을 노출하지 않고 초기화 합니다.
        DEXT5.resetEditor(true, true, editorID);
    }

</script>
