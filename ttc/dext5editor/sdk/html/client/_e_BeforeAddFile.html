DEXT5 Editor :: dext_editor_before_add_file_event

boolean dext_editor_before_add_file_event(fileInfo, pageKind, editorId)

다이얼로그에서 파일 추가 시 발생합니다.

return value

true를 리턴할 경우 파일이 추가되고, false를 리턴할 경우 파일이 추가되지 않습니다.

parameters

fileInfo  추가된 파일의 정보가 들어있는 오브젝트 입니다.
pageKind  이벤트가 발생한 다이얼로그의 종류 입니다.
editorId  이벤트가 발생한 에디터 id입니다.

remarks

해당 이벤트에서 추가된 파일 정보 또는 다이얼로그 종류를 바탕으로
파일 추가여부의 분기처리등이 가능합니다.

pageKind 종류는 아래와 같습니다.
image  이미지 삽입/수정(PC)
m_image  이미지 삽입/수정(모바일)
doc_bg_image  문서 배경 이미지
table_bg_image  표 삽입 배경 이미지
p_table_bg_image  표 속성 배경 이미지
cell_bg_image  셀 속성 배경 이미지
hyperlink_image  하이퍼링크 이미지
input_image  이미지 버튼
media  영상 삽입
flash  플래시 삽입
video_media  비디오 태그 삽입 (PC/모바일 동일)
insert_file  파일 삽입
file_open  파일 열기
