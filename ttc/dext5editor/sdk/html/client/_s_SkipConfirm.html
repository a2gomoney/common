DEXT5 Editor :: SkipConfirm

XML : <skip_confirm><new_doc>.....</new_doc></skip_confirm>, Script : SkipConfirm.NewDoc
XML : <skip_confirm><template>.....</template></skip_confirm>, Script : SkipConfirm.Template

Config 설정이 우선 적용됩니다.
새 문서, 템플릿 선택 시 confirm 사용 여부를 설정합니다.

remarks

new_doc  기본값은 "0"이고 "1"로 설정 시 새 문서 동작에서 confirm 없이 동작됩니다.
 template  기본값은 "0"이고 "1"로 설정 시 템플릿 동작에서 confirm 없이 동작됩니다.

sample code

새 문서, 템플릿 선택 시 confirm 없이 동작되도록 설정합니다.

XML

.....
<setting>
    .....
    <skip_confirm>
        <new_doc>1</new_doc>
        <template>1</template>
    </skip_confirm>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.SkipConfirm = {
            NewDoc: "1",
            Template: "1"
        }

        new Dext5editor("editor1");

    </script>       
</div>
